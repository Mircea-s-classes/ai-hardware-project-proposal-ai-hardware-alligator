Due to GitHub file size limitations, the full dataset used for training and evaluation is not included directly in this repository.

## Dataset Description
The dataset consists of bird audio recordings exported using a bird sound classification tool.  
It is organized into training and testing splits, each accompanied by a corresponding label file.

Directory structure after extraction:

data/
├── training/
│   ├── *.wav
│   └── labels.csv
├── test/
│   ├── *.wav
│   └── labels.csv
